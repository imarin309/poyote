# poyote

入力した動画から、切り抜きたい画像を選択するアプリ。

## 動画データについて

読み込んだ動画ファイルは外部サーバーへ送信されません。すべての処理はブラウザ内で完結します。

## 起動方法

```bash
pnpm install
pnpm run dev
```

`http://localhost:5173` をブラウザ（最新版のChromeを推奨）で開く。

### 動画の読み込み・再生

- 動画ファイルのドラッグ＆ドロップによる読み込み
- ファイル選択ダイアログからの読み込み
- 動画の再生（ファイル名の表示）
- 別の動画への変更
- 動画以外のファイルや再生できないファイルを選んだ場合のエラー表示

### 前後への細かい移動

- ±10秒、±1秒、±0.1秒での移動ボタン
- 再生・一時停止ボタン
- 現在時刻・動画時間の表示
- キーボードショートカット
  - `Space`：再生・一時停止
  - `←` / `→`：1秒戻る・進む
  - `Shift + ←` / `Shift + →`：0.1秒戻る・進む
  - `Alt + ←` / `Alt + →`：10秒戻る・進む
  - テキスト入力中は発火しない

### フレームの保存

- 現在表示中のフレームを元動画の解像度でWebP画像として保存（品質固定0.85）
- ベースファイル名の入力（デフォルトは動画ファイル名）
- ファイル名は `{ベースファイル名}-{分-秒-ミリ秒}.webp`
- `S`キーでの保存
- 直前に保存した画像のプレビュー表示


## コマンド

```bash
pnpm run dev      # 開発サーバー起動
pnpm run build    # 本番ビルド
pnpm run test     # テスト実行
pnpm run lint     # ESLintによる静的解析
pnpm run format   # Prettierによるフォーマット
```
